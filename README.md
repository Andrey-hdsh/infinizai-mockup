Пуш изменений
После того как вы сделали изменения в проекте:

Соберите проект:
npm run build

Зафиксируйте изменения и отправьте их в репозиторий:
git add .
git commit -m "Update project"
git push origin main

Обновите GitHub Pages:
npm run deploy
